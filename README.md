# 📘 Treinamento Swift

## 🚀 Nivelamento - Lógica e Algoritmos

Este repositório contém exercícios práticos para reforçar conceitos fundamentais de lógica de programação utilizando Swift.

---

## 📂 Conteúdo

- 🔀 Condicional e Estrutura de Escolha
- 🔁 Laços de Repetição
- 📦 ArrayList (Coleções)

---

## 🧠 Exercícios

### 🔀 Condicional e Estrutura de Escolha

1. Solicite nome, idade e cidade, e concatene em uma frase.
2. Solicite dois números inteiros:
   - Se forem iguais → exiba a soma  
   - Caso contrário → exiba a multiplicação
3. Informe um ano e retorne se é bissexto ou não.
4. Solicite três números inteiros distintos e retorne o menor.
5. Solicite um número inteiro e retorne:
   - Sucessor  
   - Antecessor  

---

### 🔁 Laços de Repetição

6. Solicite um número inteiro e exiba a tabuada.
7. Solicite nomes até que o usuário digite "sair":
   - Exiba a quantidade de nomes registrados
8. Solicite nome e duas notas:
   - Enquanto o nome for diferente de "sair"
   - Ao final, exiba o aluno com maior média
9. Solicite dois números e percorra o intervalo:
   - Quantidade de números  
   - Quantidade de pares  
   - Quantidade de ímpares  
   - Média geral  

---

### 📦 ArrayList (Coleções)

10. Solicite cinco números distintos:
   - Exiba em ordem crescente
11. Desenvolva um sistema de agenda com:
   - Nome  
   - E-mail  
   - Telefone  

   Funcionalidades:
   - ➕ Cadastrar  
   - 📋 Listar  
   - ❌ Remover  

12. Solicite cinco nomes:
   - Exiba os nomes  
   - Não permitir repetição (validar entrada)
